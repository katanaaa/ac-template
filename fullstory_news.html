Robert Pattinson is Up for Another Twilight Movie

Robert Pattinson is interested in appearing in a possible Twilight movie spin-off. The Twilight star has revealed that he would actually consider returning to the franchise if Summit Entertainment planned on expanding it. Chatting to Yahoo Movies, the actor confessed he's tempted by anything that would 'subvert' audience expectations, which we reckon a return to Twilight would definitely do. When asked if he would be up for a return to the Cullen role as the studio looks to make more Twilight movies. "Really, they’re expanding it? So I’ll get my own spin-off?" Rob enthused.
